ClaudeHUD

A macOS menu bar app that puts Claude at your fingertips. Chat, run tools, render math -- without leaving your workflow.

View on GitHub Download

Features

Everything you need from Claude, one click away.

☷

Multi-Tab Conversations

Open multiple chats with Cmd+T. Each tab keeps its own history and session. AI-generated tab titles keep things organized.

⚙

Model Switching

Switch between Sonnet, Haiku, and Opus on the fly. Pick the right model for the task at hand.

∑

KaTeX Math Rendering

Display math blocks rendered with KaTeX for publication-quality equations. Inline math uses Unicode conversion.

✎

Full Markdown

Headings, code blocks, tables, lists, task lists with checkboxes, blockquotes, and horizontal rules -- all rendered natively.

🔧

Tool Calls

See what tools Claude is using in real time. Expandable cards show arguments, results, and status.

⚙

MCP Servers

Connects to any MCP servers in your Claude environment. Use custom tools and integrations seamlessly.

Permission Modes

Control how much autonomy Claude has.

Plan

Claude proposes a plan and asks before taking any action. Maximum oversight.

Safe

Claude reads freely but asks permission before running Bash commands. The default balance.

Unsafe

No permission checks. Claude runs any command without asking. Use at your own risk.

Getting Started

Up and running in under a minute.

  • ✓ macOS 14.0 or later
  • ✓ Claude CLI installed
  • ✓ Fira Sans & Fira Code fonts (optional, falls back gracefully)
# Clone and build
git clone https://github.com/bbdaniels/ClaudeHUD.git
cd ClaudeHUD
xcodebuild -scheme ClaudeHUD -configuration Release build

# Or just open in Xcode
open ClaudeHUD.xcodeproj
